Table Tag In HTML

Table Contain - Row and Column
row - horizontal lines
col = vertical lines
Top most Row is - Table header
<tr> Tag is used to create rows in the table
<td> Tag is used to display the content of the table
<th> Tag is used to display the header of the table

Symantics in Table

1 - <thead> - To wrap table header
2 - <tbody> - To wrap table body
3 - <tfoot> - To wrap table footer
