Positioning of the Elements in the Page

It tell that what will be the positional of an element in the document.

    These are of Four Types:
  • static
    • it is the default value (by Deafault)
    • top left right bottom z-index have no effect
  • relative
    • offset are relative to itself.
  • absolute
    • Remove the element from the normal doc flow .
    • relative to it's closest positioned ancesstor. otherwise placed relative to the doc(body).
    • In the document no space is created for absolute
  • fixed
    • fixed is alway fixed in the page and relative to the initial block set by viewport.
